# php-bootstrap-waveCafe-crud

## Description
CRUD operatians using php and bootstrap

## Features

1. **Admin Panel:**
    - Insert User & List All Users
    - Insert Category & List All Categories
    - Insert Beverage & List All Beverages
    
2. **Home Page (Drink Menu):**
    - List Beverages According to Category Selected


### Admin Panel

- **User Management:**
    - Registration
    - Login
    - Logout
    - Edit User
    - Delete User

- **Category Management:**
    - Edit Category
    - Delete Category

- **Beverage Management:**
    - Edit Beverage
    - Delete Beverage

- **Additional Features:**
    - List Special Items

## Technologies Used
- Frontend: Bootstrap
- Backend: PHP
- Database: MySQL
